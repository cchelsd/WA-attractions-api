Overview

The Washington Attractions API is a comprehensive tool that provides access to a diverse array of attractions in Washington state. Divided into two core services, this API provides details, multimedia content, and tailored queires to enrich the user experience in discovering and understanding Washington attractions.

Key features

  • Comprehensive Access: Access a wide range of attractions in Washington state through a variety of tailored queries.
  • Detailed Insights: Gain in-depth information about attractions, which may include historical significance, architectural features, and multimedia content.
  • Enchanced Exploration: Facilitate exploration via random discover, highlights, proximity-based recommendations, and similarity queries.
  • Multimedia Integration: Access collections of photos and videos showcasing the attractions.

Attractions Explorer: Attractions Information

This service provides comprehensive access to attractions in Washington state, offering a variety of endpoints to discover attractions by city, category, or unique features like must-visit highlights or underrated hidden gems. It also facilitates exploration through random selections and nearby location queries. The primary focus is on retrieving information about tourist attractions based on various criteria.

Attractions by City

Provides a collection of attractions in the specified city.

Get

http://localhost:3000/attractions/city/{city}

Parameter

FieldTypeDescriptionValid Values
city text name of the city Notes:
  • When entering a value of multiple words, enter a '-', '_', or a space between words (e.g. Port Angeles or Port_Angeles, etc., PortAngeles is not valid).
  • Case insensitive, i.e. 'port Angeles' or 'Port angeles' are both valid.

Example Request and Response

http://localhost:3000/attractions/city/Seattle

Similar Attractions

Provides a collection of attractions similar to the specified attraction (given by ID). ID's are provided in request responses. Use 'All Attractions' endpoint to find ID if needed.

Get

http://localhost:3000/attractions/similar/{attraction_id}

Parameter

FieldTypeDescriptionValid Values
attraction_id int the ID of the attraction A valid ID from the database. Make a request to the All Attractions endpoint to see all of the available attractions, and their unique ID's in the database.

Example Request and Response

http://localhost:3000/attractions/similar/46 (46 corresponds to the Space Needle)

Nearby Attractions

Provides a collection of attractions near the specified location.

Get

http://localhost:3000/attractions/nearby/{location}

Parameter

FieldTypeDescriptionValid Values
location text the location of where you want to find nearby attractions Locations within Washington (state). It can be a city, county, zip code, area, address, building, etc. For example, Tacoma, King County, 98003, Space Needle, and 1900 Commerce St, are all valid values.
Notes:
  • When entering a value of multiple words, enter a '-', '_', or a space between words (e.g. Federal Way or Federal_Way, etc., FederalWay is not valid).
  • Case insensitive, i.e. 'seattle' or 'Seattle' are both valid.

Example Request and Response

http://localhost:3000/attractions/nearby/98003

Attraction Insight: Attraction Details & Media

This service offers comprehensive details about individual attractions in Washington, providing insights into topics such as their history, architecture, notable events. Other information such as contact info and addresses are offered as well. It also enriches exploration by showcasing nearby attractions and similar points of interest while offering a visual journey through a collection of attraction photos and videos.

Root Endpoint

http://localhost:3000/attractions/{attraction_id}

Endpoints

Service EndpointHTTP MethodFunctionality
http://localhost:3000/attractions/{attraction_id}
GETProvides basic info (data from Attractions Information service) about a specified attraction.
PUTUpdate basic info for the specified attraction.
POSTRemoves the specified attraction from the database.
http://localhost:3000/attractions/{attraction_id}/details
GETProvides comprehensive details about a specific attraction such as the contact info, address, history and notable events.
PUTUpdate the details of a specific attraction.
POSTAllows to add an attraction to the collection of attraction details.
http://localhost:3000/attractions/{attraction_id}/details/nearbyGETProvides a collection of details of attractions near the specified attraction.
http://localhost:3000/attractions/{attraction_id}/details/similarGETProvides a collection of details of attractions similar to the specified attraction.
http://localhost:3000/attractions/{attraction_id}/photos
GETProvides a collection of photos for a specific attraction.
PUTUpdate/add to the already existing photos for a specific attraction.
POSTAllows to add photos for a specific attraction.
http://localhost:3000/attractions/{attraction_id}/videos
GETProvides a collection of videos for a specific attraction.
PUTUpdate/add to the already existing videos for a specific attraction.
POSTAllows to add videos for a specific attraction.
